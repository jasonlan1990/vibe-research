## 📖 简介

Vibe-Research 是一个面向普通投资者的 AI 投研信息聚合平台。通过 AI 技术从多来源跨语言抓取、翻译、提炼投资资讯，降低投研门槛。

## 🌐 在线预览

👉 [https://jasonlan1990.github.io/vibe-research](https://jasonlan1990.github.io/vibe-research)

> 如链接无法访问，请先在仓库 Settings > Pages 中启用 GitHub Pages。

## ✨ 功能特性

| 功能模块 | 说明 |
|----------|------|
| 📊 **每日复盘** | AI 总结今日市场全貌，提炼核心要点 |
| 📡 **资讯雷达** | 108+ 公开源聚合，AI 跨源捞资讯、提炼要点 |
| 🗂️ **板块中心** | 追踪人形机器人、AI算力、HBM、光互联、商业航天、生物医药等热门赛道 |
| 🔍 **个股数据** | 个股行情查询 + AI 基本面分析 |
| 💼 **我的持仓** | 投资组合管理 + AI 智能诊断 |
| 📝 **研究记录** | 研究笔记沉淀，构建知识体系 |
| ⚙️ **接入AI** | 与 AI 投研助手对话，获取深度分析 |

## 🎨 设计特点

- **深色主题**：专业金融科技风格，护眼舒适
- **卡片式布局**：信息层级清晰，重点突出
- **响应式设计**：支持桌面端和移动端自适应
- **纯前端实现**：无需后端，直接部署到 GitHub Pages

## 🚀 本地运行

```bash
# 克隆仓库
git clone https://github.com/jasonlan1990/vibe-research.git

# 进入目录
cd vibe-research

# 使用任意本地服务器打开
# 方式1：Python
python -m http.server 8080

# 方式2：Node.js
npx serve .

# 方式3：VS Code Live Server 插件
```

然后在浏览器中访问 `http://localhost:8080`

## 🛠️ 部署到 GitHub Pages

1. 打开仓库页面：[https://github.com/jasonlan1990/vibe-research](https://github.com/jasonlan1990/vibe-research)
2. 点击 **Settings** 标签
3. 左侧导航选择 **Pages**
4. **Source** 选择 **Deploy from a branch**
5. **Branch** 选择 **main**，文件夹选择 **/(root)**
6. 点击 **Save**
7. 等待几分钟，访问 `https://jasonlan1990.github.io/vibe-research`
